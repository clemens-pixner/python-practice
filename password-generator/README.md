# Password Generator

A simple command-line password generator written in Python.

This project was built to practice:
- basic Python syntax
- user input handling
- loops and conditions
- the `random` module

## Features

- Generate random passwords
- Choose password length
- Uses letters, numbers, and symbols
- Runs in the terminal

## How it works

The program randomly selects characters from predefined character sets
and combines them into a password of the desired length.

## Requirements

- Python 3.x

No external libraries required.

## Usage

Run the script in the terminal:

```bash
python main.py
